NFL-Data
==========

NFL-Data seeks to provide a simple, user-friendly and developer-friendly implementation of popular NFL data analytics tools ported for Python. 

Features
------------

- Simple pre-defined data queries
- Developer sandbox
- Data visualization inclusion; including pre-made functions for the early developer

Usage
------------

NFL-Data is completly hosted online so there is no need for any installation effort.
